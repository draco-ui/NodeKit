## Variant

A button can have different visual styles to indicate hierarchy and importance.

- `primary`: emphasizes the button as a primary action
- `secondary`: indicates a secondary action
- `tertiary`: de-emphasizes the button for less important actions
- `ghost`: minimal styling that blends into the background
- `amber`: highlights the button with an amber color for warning actions
- `error`: indicates a destructive or error action
