Input fields allow users to enter text into a UI.

## Variant

An input can have different visual styles.

- `outline`: default style with a border
- `filled`: filled background style
- `underline`: minimal underline style

An input can have a 3D depth/shadow effect. The shadow can be offset to the right (default) or centered.
