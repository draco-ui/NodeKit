## Getting Started

### Installation

```bash
# React
npm install @dracoui/react

# Vue
npm install @dracoui/vue

# Ember
npm install @dracoui/ember
```

### Usage

```tsx
import { Button } from '@dracoui/react';

function App() {
  return (
    <Button variant="primary" size="medium">
      Click me
    </Button>
  );
}
```

The styles are automatically included - no need to import CSS separately!
